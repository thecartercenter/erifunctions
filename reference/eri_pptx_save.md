# Save an ERI PowerPoint to disk

Save an ERI PowerPoint to disk

## Usage

``` r
eri_pptx_save(pptx, path)
```

## Arguments

- pptx:

  An `officer` rpptx object.

- path:

  Character; output file path (should end in `.pptx`).

## Value

`path` invisibly.

## Examples

``` r
if (FALSE) { # \dontrun{
eri_pptx_save(pptx, "outputs/malaria_report.pptx")
} # }
```
