# Build one row of survey_status for a known project + form

Build one row of survey_status for a known project + form

## Usage

``` r
.odk_survey_status_row(creds, project_id, project_name, form_id)
```

## Arguments

- creds:

  Named list with `url` and `auth` from
  [`.odk_creds()`](https://thecartercenter.github.io/erifunctions/reference/dot-odk_creds.md).

- project_id:

  Integer project ID.

- project_name:

  Character project display name.

- form_id:

  Character form ID.
